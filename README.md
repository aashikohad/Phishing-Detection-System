# Adaptive Phishing Detection System

A rule-based cybersecurity system that detects phishing emails using behavioral signals and content analysis.

## Features

- Rule-based phishing detection engine
- Device anomaly detection
- Login time anomaly detection
- SOC-style monitoring dashboard
- Threat analytics and security logs

## Tech Stack

Python  
Flask API  
Streamlit Dashboard  
SQLite Database  

## Architecture

User Activity → Rule Engine → Risk Score → Security Dashboard

## Installation

Clone repository

git clone https://github.com/aashikohad/Phishing-Detection-System.git

Install dependencies

pip install -r requirements.txt

Run backend

python app.py

Run dashboard

streamlit run dashboard.py
